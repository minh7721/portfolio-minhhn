PDF Conversion SaaS

Microservices OCR Queue Jobs

Built a suite of PDF manipulation tools (Split, Merge, Compress, OCR) designed for high reliability. The challenge was processing large files (up to 500MB) without timing out the web server.

I designed an asynchronous processing pipeline. Users upload files to MinIO S3, a job is pushed to Redis, and separate worker servers pick up the heavy lifting (OCR/Conversion), notifying the user via WebSockets when complete.
